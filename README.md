## ðŸŒ How to Deploy on GitHub Pages

1.  **Initialize Git**:
    ```bash
    git init
    git add .
    git commit -m "Initial commit"
    ```
2.  **Create Repository**: Create a new public repository on GitHub named `ankitmalakar.github.io` (or any name, but this specific name enables the root domain).
3.  **Push Code**:
    ```bash
    git remote add origin https://github.com/ankitmalakar/ankitmalakar.github.io.git
    git branch -M main
    git push -u origin main
    ```
4.  **Enable GitHub Pages**:
    *   Go to **Settings** > **Pages**.
    *   Under "Source", select `main` branch.
    *   Click **Save**.
    *   Your site will be live at `https://ankitmalakar.github.io`.
